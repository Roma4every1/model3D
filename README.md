# Well Manager

### Архитектура
За основу взят архитектурный паттерн [FSD](https://feature-sliced.design).

```text
src/
├── app/                    # Инициализирующая логика приложения
├── widgets/                #
|    ├── presentation/      #   Презентация (контейнер для форм)
|    ├── top-panel/         #   Верхняя панель
|    ├── left-panel/        #   Левая панель
|    └── right-panel/       #   Правая панель
├── features/               #
|    ├── table/             #   Таблица
|    ├── chart/             #   График
|    ├── carat/             #   Каротажная диаграмма
|    ├── map/               #   Карта
|    └── multi-map/         #   Многоформенная карта
├── entities/               #
├── shared/                 # Переиспользуемые сущности
└── index.tsx               # Подключение и рендеринг приложения
|
└── assets/                 # Статические файлы
     ├── images/            #   Изображения
     └── map-libs/          #   Файлы для отрисовки карт
```
